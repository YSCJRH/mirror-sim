# Decisions

Use this directory for ADR-style records whenever a change touches long-lived project contracts.

## Must Record

- Domain schema changes
- Claim label changes
- Scenario DSL changes
- Simulation action-set or turn-model changes
- License strategy changes
- New critical external dependencies

## ADR Template

```md
# ADR-XXXX: <title>

## Status
- Proposed | Accepted | Superseded

## Context
- What changed
- Why now

## Decision
- Chosen approach

## Consequences
- Upside
- Tradeoffs
- Follow-up work
```
