# Human Review Rubric

Use the following 1-5 scoring guide when reviewing a run or report.

| Dimension | 1 | 3 | 5 |
|---|---|---|---|
| Usefulness | Adds no new understanding | Some useful contrast | Clearly clarifies branch differences |
| Credibility | Reads like guesswork | Partly grounded | Evidence boundaries are clear |
| Explainability | Hard to trace | Mostly understandable | Easy to replay from trace |
| Actionability | No next step is obvious | Some follow-up hints | Clear next engineering/product step |
